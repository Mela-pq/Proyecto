**Proyecto Integrador - Aplicación Web de Contenido Multimedia**

¡Este es el repositorio oficial de nuestro Proyecto Integrador! 
La aplicación desarrollada es una plataforma web diseñada para ofrecer una experiencia interactiva, optimizada y moderna, permitiendo a los usuarios interactuar con contenido visual y gestionar perfiles de manera dinámica.

**- Tecnologías Utilizadas -**

El proyecto está dividido en una arquitectura (Frontend y Backend) utilizando la siguiente lógica.

**Frontend**
* **React (Vite):** Librería principal para la construcción de la interfaz de usuario basada en componentes.
* **Tailwind CSS:** Framework de CSS utilitario para un diseño responsivo, moderno y estilizado.
* **React Router Dom:** Gestión de rutas y navegación SPA (Single Page Application).

**Backend**
* **FastAPI:** Framework de Python de alto rendimiento para la construcción de la API REST.
* **SQLite:** Base de datos relacional ligera utilizada para la organización de datos (usuarios, publicaciones, etc.).
* **Uvicorn:** Servidor para ejecutar la aplicación de FastAPI.
